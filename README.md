Public package containing shared static assets for Plasius frontend applications.


## Install

```bash
npm install @plasius/sharedassets
```

## Usage

```ts
import { PlasiusLTDLogo } from "@plasius/sharedassets";
```

## Development

```bash
npm install
npm run build
npm test
```
